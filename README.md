This repository contains automated backups of transcription content from [Princeton Geniza Project](https://geniza.princeton.edu/) (PGP). When possible, automated updates include co-author information in the git commit message, in order to document and credit the people responsible for the changes.

Files in these directories should *NOT* be edited on GitHub.

## Contents
- the [annotations](annotations) directory contains backups of transcription content in the W3C annotation format used internally in the PGP application
- the numeric directories include text and html files which can be found based on the PGPID of the document.
