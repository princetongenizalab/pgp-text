  11. וקדמו אהביו / וערבו עלי מאמריו / ונעמו לי דבריו / והיו אצלי
  12. כדבש למתוק בפי / וכצמיד באזרועי וכפי / והודיתי לצור
  13. ישראל על השפתת שלומותיו / והשפעת טובותיו / ברוך
  14. אשר לא ייבושו קויו / לעולם אחי ואור עיני לא הזכרתי
  15. קצת התשוקה / אלא ויש אצלי כפלי כפלים / ולא מעט
  16. מכסיפה / אלא ויש הנה שבעתים / רוכב בעזרו שמים /
  17. בורא כל ניב שפתים / יברכהו כנתברך בברכת טללי
  18. שמים / וברכות רחם ושדים / וברכות ארץ נחלי מים /
  19. וכנתברכו מקהלות יהודה ואפרים / סרגלתי אילו הטורים /
  20. נכסף אל ראייתו / ומברך אותו בברכת זה החג ושמחתו /
  21. חג השבועות ברום מעלתו / שבו נתן צורנו תורתו /
  22. לאיומתו ועדתו / יזכהו אינו שנים רבות בששונים / לחוג פני
  23. אדון האדונים / בשי ומנחה / ונסו יגון ואנחה / אעלמך יא אכי איני
  24. עולת אן שא אללה עלי אלמוגי ואכדת ברי גייר בראת בנים
  25. לאנה איצא יטלע אלי מצר וקד קבצת יז דינאר ואעלמך
  26. איצא לאני קד כתבת פי צור כתבא כתירי ובעתתהום
  27. מע אברהם אבן אל קריץ ולך איצא ובעתת מע אברהם
  28. אלמהדוי כתאבך וכתב מני וערפתה בכלאצי ואל חמד
  29. ללאה ואנא בכייר פי עאפיה מא עדִמת מן אללה כייר
  30. בסעדתך ולקד אתגעת יום אנחדארי מן אלמרכב
  31. חתי מא חסאבי אעיש ומא כתבת אולאיך אלכתב
  32. אלא מן אלצרורה ועשה למען שמו ביום חמישי
