Judith Olszowy-Schlanger, Karaite Marriage Contracts from the Cairo Geniza (Boston: Brill, 1997).

Recto

  1. [נה]ר פישון מושבה ביום הזה בא …
  2. … [ל]שון שלזכות שאני אומר לפני[כם] …
  3. … נ'נ' להיותה לי לאשה על ט[הרה ו]קדושה …
  4. … [ולא אגרע ממנה] שארה כסותה ועונתה כדי כוחי והשגת יד[י ו]אכבד
  5. … עושים כל הראוי להן באמונה ויושר המוהר
  6. … שכתוב […] את החמש[ים] הכסף אשר ה[ם מהר בתוליה] …
  7. … [לפ]ני הזקנים העדים ב[כת]ב הזה ויאמרו [א]נחנ[ו] עד[ים] …
  8. … [ע]סקי נשואי[ה] למשלם החתן הזה בן …
  9. … כתובתה ויבא הזקן יעקב בן ישעיה אשר
  10. …ה במוהר הנזכר למעלה והוא חמשים כסף מוהר
  11. … כל שבנות ישראל הכשרות עושות בבתי
  12. … הזה […] מוהר המקדם והוא עשרה דרכמונים וינקה [מ]שלם
  13. … כמסה קטע ב[ל]ור ב' אתני עשר קטעה פצ'ה נרגסיה מש[בך] [ו]מרפע
  14. …מי במסנד בגיר מכדתין ל' דר[כמון מר]תבתין טברי טברסתאן גדד בגיר
  15. … רדה מדה[ב]ה ה' ש[עריה] ה' מכדה דביקי ה' מנדיל סקל[אטו]ן
  16. …[עש]רה כלל ערך [ה]נכס[ים ה]א[ל]ה מאתי[ם ו]שבעה ושבעים דרכמון
  17. … ככל אשר לו ולא ישנה [מהם] מאומה בלא
  18. … יבלה [מ]הם וכי רצו משלם החתן בן
  19. … [הכלי]ם האלה בלתי אשר
